# Mini Game - Jeu de Saut

Un petit jeu de saut simple développé en HTML, CSS et JavaScript.

## Description

Ce mini-jeu est un projet ludique créé pour s'amuser avec les concepts de base du développement web. Le joueur doit faire sauter un personnage pour éviter les obstacles qui arrivent de la droite.

## Fonctionnalités

- **Personnage sautant** : Le personnage peut sauter pour éviter les obstacles
- **Obstacles mobiles** : Des obstacles se déplacent de droite à gauche
- **Détection de collision** : Le jeu détecte quand le personnage touche un obstacle
- **Système de score** : Le score augmente automatiquement pendant le jeu
- **Affichage du score en temps réel** : Le score s'affiche avec le même style que les boutons
- **Interface simple** : Contrôles basiques avec un bouton de saut
- **Redémarrage automatique** : Le jeu peut être redémarré avec le bouton "Start ... !"

## Comment jouer

1. Le jeu démarre automatiquement avec un score de 0
2. Cliquez sur le bouton "Jump", espace ou sur la page(dans le vide) pour faire sauter le personnage
3. Évitez les obstacles qui arrivent de la droite
4. Le score augmente automatiquement tant que vous survivez
5. Si vous touchez un obstacle, le jeu s'arrête et affiche votre score final
6. Cliquez sur "Start ... !" pour recommencer une nouvelle partie
